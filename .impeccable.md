# Impeccable Design Context — 여월농장

> 모든 UI 작업은 이 문서를 참조한다. 새로운 컴포넌트·페이지·스타일 변경 전에 반드시 읽고 시작할 것.
> 마지막 수정: 2026-05-18 (컨셉 재정의: "흙냄새 미니멀" → "일상 속 농장")

---

## Design Direction: **일상 속 농장 (Everyday Farm)**

여월농장은 1년에 단 한 번 가는 곳이 아니다. **지하철·버스로 가볍게, 평일 저녁이나 반나절에도 들러서 흙을 만지고 가는 곳**이다. 사용자는 SaaS 랜딩에 지친 도시인이지만, 동시에 "촌스러운 농장 홈페이지"에도 마음을 못 준다. 우리가 만드는 화면은 그 사이의 골짜기에 있다:

**도심 카페의 산뜻함 × 동네 농장의 정직함**.

핵심은 네 가지:
- **가볍게(Light)** — 진지한 약속이 아니라 가벼운 일상. 활자·여백·모션이 무겁지 않다.
- **가까이(Near)** — 멀고 특별한 곳이 아니라 가까운 일상. "차가 없어도 올 수 있다"가 메시지의 척도.
- **신선하게(Fresh)** — 산뜻한 종이톤·잎의 채도를 한 단 들어올린 그린. 하지만 SaaS 네온 그린은 아니다.
- **정직하게(Honest)** — 영업 문장·과장된 카피·이모지·그라데이션 텍스트로 채우지 않는다. 운영자가 직접 손으로 적은 것 같은 절제.

## Users

### Primary: 20-50대 수도권 도시 거주자 (확장)
가족·솔로·커플·직장인을 **동등한 무게**로 다룬다. 5평 한 구획이 4인 가족에게도, 부부 둘에게도, 친구 셋에게도 의미 있게 보여야 한다.

- **연령·구성**: 20-50대. 아이와 함께 자연을 경험하고 싶은 가족, 멀리 가지 않고 초록을 보고 싶은 직장인·커플 모두 포함.
- **거주지**: 부천·서울 서남부·인천 일부. **차량 없음 시나리오를 1순위**로 디자인한다(지하철·버스 안내 비중 가족과 동등).
- **컨텍스트**: 신청 시즌(1월 초)에 노트북 또는 모바일로 사이트 방문. 가족 회의 후 결정하는 경우, 친구·연인과 같이 보는 경우 모두 있음. 결정 사이클은 **짧다** — "이거 한번 해볼까?" 정도의 가벼운 무게.
- **잡(JTBD)**:
  1. "여월농장이 어떤 곳인지 5분 안에 파악 — 멀지 않고, 차 없어도 갈 만한 곳인지"
  2. "5평이 우리(가족/커플/혼자)에게 적당한지 판단"
  3. "신청 절차가 깔끔하고 정직한지 확인"
  4. "신청 후 내 구획·계좌·일정을 사이트에서 확인"

### Secondary: 호정 농업회사법인 운영자 (어드민)
- **연령**: 50~70대. 모바일·태블릿 중심 사용.
- **컨텍스트**: 주 2~3회 어드민 접속. 신청 검토 → 입금 확인 → 배정 → 공지 작성.
- **잡**: 빠르고 실수 없이. 화려한 인터랙션·애니메이션·툴팁 폭격은 방해.

## Brand Personality

### 4-word personality
**가볍게 · 가까이 · 신선하게 · 정직하게**

- **가볍게(Light)**: "1년에 단 한 번 신중한 결정"이 아니라 "반나절 짬 내서 들르는 곳". 활자·여백·CTA 모두 무게를 덜어낸다.
- **가까이(Near)**: 도심에서의 거리감을 디자인 전반에 녹인다. 지하철·버스 안내, 부천 위치, 30분 도착이 메시지의 중심.
- **신선하게(Fresh)**: 잎의 채도를 살짝 올린다(SaaS 네온은 절대 아님). 흙·종이톤은 유지하되 더 환하게.
- **정직하게(Honest)**: 운영자가 직접 적은 안내문 톤. 영업 문구·과장·이모지 없음.

### Voice & Tone
- **친근한 존댓말**. "신청해보세요" 정도는 OK, "지금 바로 신청하세요!" "놓치지 마세요!" 영업체 금지.
- 숫자는 작은 글자로 숨기지 않고 본문 활자로 당당하게 노출.
- "AI 보정", "최첨단", "프리미엄" 같은 미사여구·테크 용어 사용 금지.
- 한국어 안에 영문 약어(CTA, FAQ, JTBD) 노출 최소화. 메뉴는 한국어 우선.
- **단호함은 줄이고 가벼움은 늘린다.** "한 해 약속" 같은 무거운 단어는 본문 가능, 헤드라인 X.

### Emotional goals
| 사용자가 느껴야 할 것 | 느끼면 안 되는 것 |
|---|---|
| 가벼움 ("한번 해볼까? 무겁지 않네") | 부담 ("1년 계약은 너무 무거운데…") |
| 가까움 ("의외로 가깝네, 차 없어도 되네") | 거리감 ("멀어서 못 가겠다") |
| 신선함 ("종이톤이지만 답답하지 않다") | 촌스러움 ("70년대 농가 사이트 같다") |
| 정직함 ("운영자가 직접 적은 안내문 같다") | 의심 ("이거 AI가 만든 SaaS 사이트인가?") |

## Aesthetic Direction

### Theme
**라이트 모드 단일**. 다크 모드 미지원.
이유: 주 사용 시점이 낮 시간 + 농원=햇볕·흙=라이트가 컨텍스트적으로 옳다. "도심 카페" 무드도 라이트 기본.

### Color palette (oklch)

흙·잎·종이로 만든 톤이지만 **이전보다 한 단 환하게**. 채도는 여전히 의심하고 시작하되, "신선하게" 키워드에 맞게 leaf-500을 살짝 들어올린다.

```
/* 페이퍼 (배경) — 환하게 한 단 올림 */
--paper:        oklch(0.985 0.005 95)   /* 미세하게 따뜻한 오프화이트 */
--paper-warm:   oklch(0.965 0.012 88)   /* 섹션 구분 — 황톳빛 톤 살짝 따뜻 */
--ink:          oklch(0.20 0.01 90)     /* 본문 — 순검정 금지, 약하게 따뜻 */
--ink-soft:     oklch(0.48 0.01 90)     /* 보조 텍스트 — 이전보다 한 단 밝게 */

/* 잎 (브랜드, 산뜻함 한 단 부스트) */
--leaf-50:      oklch(0.96 0.022 130)
--leaf-200:     oklch(0.86 0.055 132)
--leaf-500:     oklch(0.52 0.105 132)   /* 메인 — 채도 0.085 → 0.105, 휴 130 → 132 */
--leaf-700:     oklch(0.38 0.085 132)   /* 헤딩·강조 텍스트 */
--leaf-900:     oklch(0.22 0.055 132)

/* 흙 (액센트) — 유지 */
--soil-300:     oklch(0.72 0.06 65)     /* 따뜻한 황톳빛 — 공지·날짜·시즌 */
--soil-600:     oklch(0.50 0.09 55)     /* 시그널 액센트 */

/* 신호 (최소 사용) */
--signal-warn:  oklch(0.70 0.13 75)
--signal-stop:  oklch(0.55 0.18 28)
```

**중성색은 brand hue로 살짝 틴트**(`0.005~0.012` chroma) — 농원답게 회색이 따뜻함을 머금는다.
**순백·순흑 절대 금지**.
**채도 상한선**: leaf-500의 0.105 chroma가 천장. 이보다 채도 높이면 SaaS 그린으로 넘어간다.

### Typography

**페어링 유지, 사용감 더 가볍게**:
- 표제(Display/H1/H2): `Gowun Batang` (Google Fonts) — 둥글지 않은 본명조. 무겁게 들렸던 명조를 자간·행간·여백을 더 풀어 **'잡지 표지'가 아닌 '편안한 책방 간판'** 무게감으로.
- 본문/UI: `Pretendard Variable` — 가독성·숫자·라벨.
- 숫자(가격·계좌·신청번호): Pretendard `tabular-nums` + `tracking-tight`.

**가벼움 가이드**:
- Display/H1에 `tracking-[-0.01em]` 정도 살짝 좁히기 — 명조 단단함은 유지하되 활자 사이 공기 더.
- Display/H1 행간: `leading-[1.15]` → `leading-[1.2]` 정도로 한 단 늘림.
- 표제 위 라벨 자리에 작은 영문 약어(`SEASON`, `LOCATION`) 절대 X — 한국어로 "시즌", "위치".

**금지**:
- 모든 곳을 Pretendard 단일로 쓰지 않는다. 표제는 Gowun Batang.
- 한국어 본문에 `uppercase` 클래스 사용 금지 (무효 + 영문 SaaS 모방).
- `tracking-wider`로 자간 벌린 작은 라벨 패턴 금지.

**타입 스케일** (fluid clamp 유지):
```
--text-display: clamp(2.5rem, 5.5vw, 4.25rem)   /* Hero only */
--text-h1:      clamp(2rem, 3.5vw, 3rem)
--text-h2:      clamp(1.5rem, 2.4vw, 2.125rem)
--text-h3:      1.25rem
--text-body:    1rem
--text-meta:    0.875rem
```
단계 간 1.25 이상 비율 유지. 본문은 `--text-body` 고정, 표제만 fluid.

**라인 길이**: 본문 컨테이너 `max-w-[65ch]` 강제.

### Spacing & Layout

- 4pt 스케일 유지. 페이지 단위 **여백을 한 단 더 넓힌다**(Hero/섹션 사이 `py-16` → `py-20~24`). "가볍게" 키워드의 핵심 표현.
- **카드 안에 모든 것을 넣지 않는다**. 동네 카페 메뉴판은 카드 없이도 정보가 산다.
- **Hero `text-center` 금지**. 좌측 정렬 + 우측 1/3에 메타(시즌·연락처·날짜) 비대칭이 기본. 단, `/apply/success` 결과 페이지는 예외.
- 섹션 구분은 카드가 아니라 **여백 + 가는 1px `--leaf-200/40` 디바이더 + 종이톤 배경 교차**.
- 사진은 16:9 또는 4:5 크게. **6칸 썸네일 그리드 SaaS 패턴 금지**.

### Hero pattern (확정)

```
┌──────────────────────────────┐
│       [풀폭 농장 사진]        │
│                              │
│  표제 (Gowun Batang, 좌측)    │
│  서브 (Pretendard, 1줄)       │
│                              │
├──────────────────────────────┤
│ 시즌 │ 2027.3-11 │ 위치 부천 │   ← dl 띠
└──────────────────────────────┘
```

- 풀폭 16:9 사진 1장 (사진 위 약간 darken 오버레이 `bg-black/10~20` 정도)
- 사진 위에 좌측 정렬 표제 + 1줄 서브카피
- 사진 아래 dl 띠 — 시즌·날짜·위치 등 메타 3-4개
- article header(아이콘+제목+서브카피 정렬) 형식 사용 금지 — 검증된 패턴 아님

### Photography (최우선 자산)

- `사진/` 폴더 9장이 **이 사이트의 핵심 자산**. 미사용 = 사이트가 영원히 AI 슬롭처럼 보임.
- "동네 카페 무드"에 맞는 사진 우선 큐레이션:
  - 햇살 든 텃밭 풀숏 → Hero
  - 사람 손·도구·물뿌리개 → "체험" 섹션
  - 산책길·쉼터 → "휴식" 섹션
  - 입구·약도 → "오시는 길"
- AI 보정 톤·노출 보정 수준까지. 비현실적 채도·HDR 금지.
- 그라데이션 placeholder는 한 군데도 남기지 않는다.

### Texture (절제)

- 종이 텍스처는 **선택적**. 추가한다면 SVG noise 5~8% opacity, 배경 한 군데(Hero 또는 풋터)에만.
- 잡지스러운 종이 결을 흉내내는 패턴 이미지·radial gradient overlay 금지.

### Iconography

- 이모지(🌱 🚰 🚗 📢 🗺️ 등) 모든 위치에서 제거.
- 아이콘이 필요하면 lucide-react의 **1px stroke** 라인 아이콘만, `--leaf-700` 단일 색.
- 시설·교통 안내 같은 곳은 아이콘 없이 텍스트만 + 좌측 정렬 dot prefix(•)로 충분.

### Motion

- 페이지 로드 시 큰 페이드 인 금지.
- 인터랙션 모션은 폼 검증(에러 inline) · 신청 완료 페이지 1회 · 본인 조회 결과 카드 1회로 제한.
- ease-out-quart, 200~280ms 이내. Bounce·spring 금지.
- 스크롤 기반 자동 페이드 인/슬라이드 인 도입 금지 — "가볍게"는 느릿한 등장이 아니라 즉시 보이는 것.

## Key Messages (사이트 카피의 원천 3문장)

새 컨셉 문서의 3대 메시지. Hero·About·시설 안내 등 헤드라인은 이 셋에서 변형해서만 나온다.

1. "**멀리 가지 않아도 자연을 느낄 수 있다**" — 도심 근접성·일상성
2. "**차가 없어도 올 수 있다**" — 교통 접근성, 도시인을 위한 약속
3. "**반나절만 내도 충분히 힐링할 수 있다**" — 가벼운 시간 단위, 부담 없는 약속

이 세 메시지가 시즌·계좌·평수 같은 사실 정보보다 **먼저** 보여야 한다.

## Design Principles (모든 결정의 척도)

1. **활자가 먼저, 장식이 나중** — 폰트 페어링·타입 스케일·여백으로 위계가 안 서면, 카드·보더·그라데이션을 더하지 말고 활자를 다시 본다.
2. **사진은 그라데이션·일러스트·이모지를 대체한다** — placeholder 그라데이션은 임시 사진보다도 나쁘다. 사진이 없으면 그 자리를 비워라.
3. **신선함은 채도가 아니라 여백에서 온다** — 색을 더하기 전에 여백을 더한다. leaf-500의 0.105 chroma가 천장이고, 그 이상은 SaaS다.
4. **카드는 마지막 수단** — 정보를 묶으려 할 때 카드부터 떠올렸다면 잘못된 출발이다. 여백·디바이더·종이톤 배경 교차로 먼저 시도.
5. **가볍게 = 즉시 보임** — "가볍게"는 느릿한 페이드 인이 아니라 빠른 도착이다. 모션·로딩·인터랙션 모두 사용자 시간을 끌지 않는다.
6. **한국어 사용자에게 정직하게** — 영문 SaaS 패턴(uppercase 라벨, tracking-wider, "Get started" 버튼)을 한국어에 옮기지 않는다. 라벨도 한국어("시즌", "위치", "신청하기").
7. **AI 슬롭 테스트 + 촌스러움 테스트** — 화면을 만든 후 두 번 묻는다: ① "AI가 만들었다고 하면 믿을까?" ② "운영자(50-70대)가 보고 '촌스럽다'고 할까?" 둘 다 No여야 통과.
