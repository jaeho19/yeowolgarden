# Impeccable Design Context — 여월농장

> 모든 UI 작업은 이 문서를 참조한다. 새로운 컴포넌트·페이지·스타일 변경 전에 반드시 읽고 시작할 것.
> 마지막 수정: 2026-05-18

---

## Design Direction: **흙냄새 미니멀 (Earthy Minimalist)**

여월농장은 SaaS 스타트업이 아니다. 30년 가까이 운영된 부천 외곽의 텃밭이다.
사용자가 1년에 단 한 번, 무겁지 않지만 신중하게 결정을 내리는 곳 — 그 결정의 무대는
"트렌디한 그린 그라데이션 랜딩"이 아니라 **종이 위에 적힌 단단한 안내문**에 가까워야 한다.

핵심은 세 가지:
- **톤 다운된 컬러** — 채도를 낮춘 흙·잎·황톳빛
- **단단한 명조 표제** — 본명조(세리프) 한국어 표제 + Pretendard 본문
- **실물 사진 중심** — 그라데이션·일러스트·이모지로 채우지 않는다

---

## Users

### Primary: 주말 텃밭 신청자
- **연령**: 35~60대 가족. 보호자(엄마/아빠 또는 은퇴 직전 부부)가 폼을 작성한다.
- **거주지**: 부천·서울 서남부·인천 일부. 차량 1대 보유 가정 비중 높음.
- **컨텍스트**: 1년에 한 번, 1월 초 신청 시즌에 노트북 또는 모바일로 사이트를 방문한다. 가족 회의 후 결정하는 경우가 많고, "정말 운영되는 곳 맞나" 의심을 즉시 해소해야 한다.
- **잡(JTBD)**:
  1. "여월농장이 어떤 곳인지 5분 안에 파악"
  2. "5평이 우리 가족에게 적당한지 판단"
  3. "신청 절차가 사기 같지 않은지 확인"
  4. "신청 후 내 구획이 어디인지 확인"

### Secondary: 호정 농업회사법인 운영자
- **연령**: 50~70대. 모바일·태블릿 중심 사용.
- **컨텍스트**: 일주일에 2~3회 어드민에 접속. 신청 검토 → 입금 확인 → 배정 → 공지 작성.
- **잡**: 빠르고 실수 없이. 화려한 인터랙션·애니메이션·툴팁 폭격은 방해.

## Brand Personality

### 3-word personality
**단단함 · 흙냄새 · 한 해 약속**

- **단단함(Solid)**: 30년 운영의 무게. 흔들리지 않는 활자, 명확한 가격, 군더더기 없는 절차.
- **흙냄새(Earthy)**: 비싼 향이 아닌 흙·잎·마른풀 냄새. 미세하게 따뜻하지만 달지 않다.
- **한 해 약속(Seasonal contract)**: 3월~11월 9개월. 가벼운 구독이 아니라 가족이 합의한 1년 단위 약속.

### Voice & Tone
- 존댓말, 단호한 문장. 영업 문체("지금 신청하세요!" "놓치지 마세요!") 금지.
- 숫자는 작은 글자로 숨기지 않고 본문 활자로 당당하게 노출.
- "AI 보정", "최첨단" 같은 미사여구·테크 용어 사용 금지.
- 한국어 안에 영문 약어(CTA, FAQ, JTBD) 노출 최소화. 메뉴는 한국어 우선.

### Emotional goals
| 사용자가 느껴야 할 것 | 느끼면 안 되는 것 |
|---|---|
| 안심 ("진짜 운영되는 곳이구나") | 흥분 ("얼른 신청해야 해!") |
| 신뢰 ("운영자가 책임감 있게 관리하는구나") | 의심 ("이거 자동 생성 사이트 아닌가?") |
| 계절감 ("올해 3월에 흙 만질 생각 좋다") | 디지털 피로 ("또 SaaS 랜딩페이지?") |

## Aesthetic Direction

### Theme
**라이트 모드 기본**. 어둠 모드 미지원(이 프로젝트에선 추가하지 않는다).
이유: 주 사용 시점이 1월 초 낮 시간 + 농원=햇볕·흙=라이트가 컨텍스트적으로 옳다.

### Color palette (oklch)

흙·잎·종이로 만든 톤 다운 팔레트. 채도는 일부러 낮춘다. 기존 `--color-brand-*`는 채도가 너무 높음(0.13~0.15 chroma) → 0.05~0.09로 낮춰 재정의 필요.

```
/* 페이퍼 (배경) */
--paper:        oklch(0.985 0.005 95)   /* 미세하게 따뜻한 오프화이트 */
--paper-warm:   oklch(0.96 0.012 90)    /* 섹션 구분용 황톳빛 종이 */
--ink:          oklch(0.20 0.01 90)     /* 본문 — 순검정 금지, 약하게 따뜻 */
--ink-soft:     oklch(0.45 0.01 90)     /* 보조 텍스트 */

/* 잎 (브랜드, 톤 다운된 그린) */
--leaf-50:      oklch(0.96 0.018 125)
--leaf-200:     oklch(0.85 0.045 125)
--leaf-500:     oklch(0.48 0.085 130)   /* 메인 브랜드 — 기존 0.55 0.14 에서 채도 낮춤 */
--leaf-700:     oklch(0.36 0.07 130)    /* 헤딩·강조 텍스트 */
--leaf-900:     oklch(0.22 0.05 130)

/* 흙 (액센트) */
--soil-300:     oklch(0.72 0.06 65)     /* 따뜻한 황톳빛 — 공지·날짜·시즌 표시 */
--soil-600:     oklch(0.50 0.09 55)     /* 시그널 액센트 */

/* 신호 (최소 사용) */
--signal-warn:  oklch(0.70 0.13 75)     /* amber, 마감 임박 등에만 */
--signal-stop:  oklch(0.55 0.18 28)     /* 환불 불가·오류에만 */
```

**중성색은 brand hue로 살짝 틴트**(`0.005~0.012` chroma) — 농원답게 회색이 따뜻함을 머금는다.
**순백·순흑 절대 금지**.

### Typography

**페어링**:
- 표제(Display/H1/H2): `Gowun Batang` (Google Fonts) — 둥글지 않은 본명조, 단단함. 농원의 손글씨 같지 않으면서도 활자의 무게가 있다.
- 본문/UI: `Pretendard Variable` (유지) — 가독성·숫자·라벨.
- 숫자(가격·계좌·신청번호): Pretendard `tabular-nums` + `tracking-tight`.

**금지**:
- 모든 곳을 Pretendard 단일로 쓰지 않는다.
- 한국어 본문에 `uppercase` 클래스 사용 금지 (무효 + 영문 SaaS 모방 느낌).
- `tracking-wider`로 자간 벌린 작은 라벨 패턴 금지. 라벨은 `leaf-700` + 일반 자간.

**타입 스케일** (fluid clamp 도입):
```
--text-display: clamp(2.5rem, 5.5vw, 4.25rem)   /* Hero only */
--text-h1:      clamp(2rem, 3.5vw, 3rem)
--text-h2:      clamp(1.5rem, 2.4vw, 2.125rem)
--text-h3:      1.25rem
--text-body:    1rem
--text-meta:    0.875rem
```
단계 간 1.25 이상 비율 유지. 본문은 `--text-body` 고정, 표제만 fluid.

**라인 길이**: 본문 컨테이너 `max-w-[65ch]` 강제. 와이드 페이지에서도 본문 단락은 좁게.

### Spacing & Layout

- 4pt 스케일 유지. `py-0.5`(2px) 같은 토큰 밖 값은 제거.
- **카드 안에 모든 것을 넣지 않는다**. 종이 위의 안내문은 카드 없이도 위계가 산다.
- **모든 Hero `text-center` 금지**. 좌측 정렬 + 우측 1/3에 메타(시즌·연락처·날짜) 비대칭이 기본.
- 섹션 구분은 카드가 아니라 **여백 + 가는 1px `--leaf-200/40` 디바이더 + 종이톤 배경 교차**.
- 사진은 16:9 또는 4:5 크게. 작은 썸네일 그리드 6칸 같은 SaaS 패턴 지양.

### Photography (최우선 자산)

- `사진/` 폴더 9장이 **이 사이트의 핵심 자산**. 미사용 = 사이트가 영원히 AI 슬롭처럼 보임.
- AI 보정은 톤 보정·노출 보정 정도까지. 비현실적 채도·HDR 금지. **흙은 흙처럼, 잎은 잎처럼**.
- 그라데이션 placeholder는 한 군데도 남기지 않는다.

### Texture (절제)

- 종이 텍스처는 **선택적**. 추가한다면 SVG noise 5~8% opacity, 배경 한 군데(Hero 또는 풋터)에만.
- 잡지스러운 종이 결을 흉내내는 패턴 이미지·radial gradient overlay 금지.

### Iconography

- 이모지(🌱 🚰 🚗 📢 🗺️ 등) 모든 위치에서 제거.
- 아이콘이 필요하면 lucide-react의 **1px stroke** 라인 아이콘만, `--leaf-700` 단일 색.
- 시설 안내 같은 곳은 아이콘 없이 텍스트만 + 좌측 정렬 dot prefix(•)로 충분.

### Motion

- 페이지 로드 시 큰 페이드 인 금지. 사용자는 1년 만에 신청하러 왔다 — 시간 끌지 않는다.
- 인터랙션 모션은 폼 검증(에러 흔들림 없이 inline) · 신청 완료 페이지 1회 · 본인 조회 결과 카드 1회로 제한.
- ease-out-quart, 200~280ms 이내. Bounce·spring 금지.

## Design Principles (모든 결정의 척도)

1. **활자가 먼저, 장식이 나중** — 폰트 페어링·타입 스케일·여백으로 위계가 안 서면, 카드·보더·그라데이션을 더하지 말고 활자를 다시 본다.
2. **사진은 그라데이션·일러스트·이모지를 대체한다** — placeholder 그라데이션은 임시 사진보다도 나쁘다. 사진이 없으면 그 자리를 비워라.
3. **채도는 의심하고 시작한다** — 컬러를 더하기 전에 "이게 톤 다운된 흙·잎·종이 안에서 의미를 가지나?"를 묻는다.
4. **카드는 마지막 수단** — 정보를 묶으려 할 때 카드부터 떠올렸다면 잘못된 출발이다. 여백·디바이더·종이톤 배경 교차로 먼저 시도.
5. **한국어 사용자에게 정직하게** — 영문 SaaS 패턴(uppercase 라벨, tracking-wider, "Get started" 식 버튼)을 한국어에 그대로 옮기지 않는다.
6. **AI 슬롭 테스트** — 모든 화면을 만든 후 "AI가 만들었다고 하면 믿을까?"를 묻는다. 믿긴다면 처음으로 돌아간다.

## When in doubt

종이 위에 손으로 적힌 안내문을 떠올린다. 거기에 그라데이션·이모지·glassmorphism이 없다면, 화면에도 없어야 한다.
