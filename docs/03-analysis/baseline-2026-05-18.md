### 5.3 측정 환경 한계 (결과 해석)

LCP 23-25초는 비현실적 — localhost 측정의 산물:

1. **Lighthouse Mobile 기본 throttling**: Slow 4G(~1.6Mbps) + 4× CPU slowdown 시뮬레이션. 4MB 자산 다운로드만 약 20초.
2. **CDN 부재**: localhost는 모든 자산을 origin에서 직접 서빙. 실 Netlify는 edge CDN에서 즉시 서빙.
3. **next/image cold cache**: 첫 요청 시 JPG → WebP 변환 캐시 빌드. 두 번째 요청부터 즉시.
4. **Bytes는 페이지 로드 총합**: first paint에 필요한 자원만이 아니라 lazy load 자원까지 포함.

→ **Netlify 프로덕션 + 사용자 디바이스 측정이 정확한 baseline**. localhost 결과는 상한선(워스트 케이스).
